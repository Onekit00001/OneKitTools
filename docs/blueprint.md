# **App Name**: OneKit

## Core Features:

- Image Compressor: Compress images to reduce file size without significant quality loss, directly in the browser.
- Image Resizer: Resize images to specific dimensions, maintaining aspect ratio where desired.
- Image Converter: Convert images between JPG, PNG, and WEBP formats.
- Add Text/Watermark to Image: Add custom text or watermarks to images to protect or brand them.
- Flip/Rotate Image: Flip images horizontally or vertically, and rotate them by 90-degree increments.
- Scientific Calculator: A fully functional scientific calculator for complex calculations.
- BMI Calculator: Calculate Body Mass Index (BMI) based on height and weight.
- Age Calculator: Determine age based on birth date and current date.
- Loan EMI Calculator: Calculate Equated Monthly Installment (EMI) for loans.
- SIP Calculator: Calculate returns on Systematic Investment Plan (SIP) investments.
- Percentage Calculator: Perform various percentage-based calculations.
- Currency Converter: Convert between different currencies using live exchange rates (via API).
- GST/Tax Calculator: Calculate Goods and Services Tax (GST) or other taxes.
- Tip Calculator: Calculate tips and split bills among multiple people.
- Date & Time Calculator: Calculate the difference between two dates or times.
- Time Duration Calculator: Calculate the duration between two timestamps.
- Discount Calculator: Calculate the final price after applying discounts.
- Fuel Cost Calculator: Calculate the cost of fuel for a journey.
- Word & Character Counter: Count the number of words and characters in a given text.
- Case Converter: Convert text to uppercase, lowercase, or title case.
- Remove Extra Spaces & Lines: Clean up text by removing extra spaces and line breaks.
- Remove Duplicate Lines: Remove any duplicate lines from a text input.
- Text to Slug: Generate a URL-friendly slug from a given text.
- List Sorter: Sort lists alphabetically or randomly.
- Morse Code Translator: Translate text to Morse code and vice versa.
- Binary <-> Text Converter: Convert between binary code and text.
- JSON Formatter & Validator: Format and validate JSON code.
- Base64 Encode/Decode: Encode and decode Base64 strings.
- URL Encode/Decode: Encode and decode URLs.
- Timestamp Converter: Convert timestamps to human-readable dates and vice versa.
- QR Code Generator: Generate QR codes with custom colors and logos.
- Advanced Color Picker: Pick colors using a pipette, view color history, save palettes, and copy colors in various formats.
- Strong Password Generator: Generate strong, secure passwords with customizable options.
- Random Number Generator: Generate random numbers within a specified range.
- Unit Converter: Convert between various units of measurement (length, weight, temperature, speed, volume, area).
- Time Zone Converter: Convert times between different time zones, including major cities.
- Countdown Timer: Set a countdown timer for specific events.
- Pomodoro Timer: A timer to use in pomodoro-style time management.
- Stopwatch: A simple stopwatch for measuring elapsed time.
- Lorem Ipsum Generator: Generate placeholder text for layouts and designs.
- Emoji Picker: Select and copy emojis easily.
- Rich Text Editor: Create and format text with basic styling options (bold, italic, lists, colors, headings), and save as HTML.
- Note-Taking App: Create, edit, delete, and search notes; saved locally in the browser.
- To-Do List: Manage tasks with drag-and-drop functionality and categorized lists; saved locally in the browser.
- World Clock: Display the current time for multiple cities around the world.
- Simple Weather Widget: Display weather information from a free weather API.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) for a modern and engaging look.
- Background color: Light blue (#E0F7FA), subtly desaturated.
- Accent color: Sky blue (#87CEEB) to highlight key elements and calls to action.
- Body and headline font: 'Inter' sans-serif for a clean and modern look. Note: currently only Google Fonts are supported.
- Code font: 'Source Code Pro' for displaying code snippets. Note: currently only Google Fonts are supported.
- Use simple, outline-style icons for a clean and modern look.
- Employ a grid-based layout with responsive design to ensure optimal viewing on all devices.
- Subtle animations for transitions and user interactions to enhance the user experience.
